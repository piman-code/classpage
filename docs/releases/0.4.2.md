# classpage 0.4.2 Beta

`classpage 0.4.2`는 닫힌 베타 / 소규모 실사용 테스트를 위한 릴리즈입니다.  
이번 버전은 기능 확장보다 배포 정합성을 우선해, 개인 운영 흔적이 제거된 최신 커밋 기준으로 prerelease 경로를 다시 정리한 베타입니다.

## 이번 베타에서 가능한 것

- 학생용 페이지
  오늘의 할 일, 공지사항, 학급용/수업용 Google Form 버튼 확인
- 선생님용 학급 대시보드
  학급 상태, 정서/목표 분포, 도움이 필요한 학생, 칭찬 후보 확인
- 선생님용 수업 대시보드
  과목 -> 단원 -> 최근 수업 범위/특정 날짜 -> 수업 그룹 탐색
- 학생 명단 기반 미제출 학생 확인
  overview, 학급, 수업 화면에서 응답이 아예 없는 학생까지 함께 확인
- 학생 명단 CSV 가져오기 도우미
  CSV 파일 선택 또는 표 붙여넣기로 `student-roster.json` 저장
- 선생님 화면 보기 기준 조정
  `기본형 / 위험 조기 발견형 / 칭찬 강화형 / 미제출 집중형` 프리셋과 정렬/강조 설정
- 수업 follow-up 확인
  다음 피드백 대상, 재설명 필요한 개념, 복습/수행 상태, 학생별 결과 확인
- 선생님용 별점 대시보드
  규칙 현황, 최근 이벤트, 학생별 누적, 학생 검색, 최근 변동 학생 흐름 확인
- 별점 자동 적립 + 시트 기반 수동 조정 반영
  Google Sheets `별점 수동 조정` 시트와 Apps Script 결과가 `star-ledger.json`에 반영되면 classpage에서 읽기 전용으로 확인 가능

## 이번 베타의 성격

- 닫힌 베타 / 소규모 실사용 테스트
- 읽기 중심 선생님 운영 대시보드
- GitHub prerelease 경로를 최신 안전 커밋 기준으로 재정리한 배포
- Google Sheets / Apps Script / JSON 집계 결과를 읽는 표시 레이어

## 아직 없는 것

- 플러그인 안에서 Google Sheets 직접 쓰기
- 플러그인 안에서 별점 수동 조정 입력
- 플러그인 안에서 규칙 편집
- AI 기능
- 복잡한 차트 시스템
- 다중 반/다중 사용자 운영 기능

## 권장 사용자

- Google Sheets + Apps Script + JSON 연결을 직접 실험해볼 수 있는 선생님
- 학급/수업/별점 상태를 한 화면에서 빠르게 확인하고 싶은 초기 사용자
- 읽기 중심 운영 흐름을 먼저 검증해 보고 싶은 닫힌 베타 참여자

## 주의사항

- 사용 전에 Google Sheets + Apps Script + JSON 연결이 필요합니다.
- 선생님 화면은 원본 응답이 아니라 집계 JSON을 읽습니다.
- 별점 수동 조정은 플러그인 안이 아니라 Google Sheets `별점 수동 조정` 시트에서 합니다.
- 일부 구형 JSON에서는 단원/날짜 정보가 부족해 필터가 `정보 없음`으로 묶여 보일 수 있습니다.

## 확인하고 싶은 포인트

- 선생님이 수업 결과를 보고 다음 피드백 대상을 빠르게 고를 수 있는지
- 학생 명단을 JSON 수작업 없이도 CSV 가져오기 도우미로 무리 없이 준비할 수 있는지
- 미제출 학생 카드가 과하지 않으면서도 실제 독촉/확인 대상 파악에 충분히 도움이 되는지
- 별점 화면에서 학생별 최근 흐름을 오해 없이 이해할 수 있는지
- 처음 연결하는 선생님이 빈 상태와 누락 상태를 덜 헷갈리는지
